# Invitecode.is

## How to run
* `yarn install` then `yarn start` to develop and `yarn build:pwa` to build it 
* Read more on the documentation of Jekyll Webpack Boilerplate: https://github.com/sandoche/Jekyll-webpack-boilerplate
